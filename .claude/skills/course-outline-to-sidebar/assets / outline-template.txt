## Instructions

1. Replace placeholders with your course content
2. Use "Week X" for single weeks or "Weeks X-Y" for multi-week sections
3. List chapters as bullet points under each week
4. Keep a clear hierarchy: Module → Week → Chapter
5. Save this file and use it with parse_outline.py:

   python3 scripts/parse_outline.py course_outline.txt

## Format Guidelines

### Module Format
- Start with "Module {number}: {Title}"
- Example: "Module 1: Introduction to Python"

### Week Format
- Single week: "Week {number}: {Topic}"
- Multi-week: "Weeks {start}-{end}: {Topic}"
- Examples:
  - "Week 1: Getting Started"
  - "Weeks 2-4: Advanced Concepts"

### Chapter Format
- Use bullet points (- or •)
- Keep titles concise and descriptive
- Example: "- Introduction to variables and data types"
