# 2.2 Robot Model Design & Integration

## Learning Objectives

By the end of this week, students will be able to:
- Import URDF and SDF robot models into Gazebo
- Configure joint properties and transmissions
- Set up robot state publishers for simulation
- Spawn custom robots in simulation environment
- Validate robot model behavior in simulation

## Topics Covered

### Day 1: URDF and SDF Basics
- Understanding URDF (Unified Robot Description Format)
- SDF (Simulation Description Format) overview
- Converting between URDF and SDF
- Model structure and components

### Day 2: Robot Model Configuration
- Joint types and properties
- Transmission definitions
- Visual and collision properties
- Inertial parameters

### Day 3: Robot State Publishing
- Robot State Publisher role in simulation
- Joint State Publisher configuration
- TF transforms in simulation
- Sensor integration basics

### Day 4: Spawning Robots in Gazebo
- Using spawn_entity with custom models
- Initial pose and configuration
- Plugin configuration for simulation
- Robot spawning scripts

### Day 5: Model Validation and Testing
- Validating robot kinematics in simulation
- Checking joint limits and ranges
- Testing basic movement and control
- Week project: Custom robot integration
